# Markdown Features Guide

This file demonstrates common **Markdown** features with examples.

## 1. Headings

Use `#` for headings:

# Heading 1

## Heading 2

### Heading 3

#### Heading 4

------------------------------------------------------------------------

## 2. Hyperlinks

Inline link:\
[OpenAI](https://www.openai.com)

Reference-style link:\
[Google](https://www.google.com)

------------------------------------------------------------------------

## 3. Embedding Images

Basic syntax:\
`![alt text](image_url)`

Example:

![Sample Image](https://via.placeholder.com/300)

------------------------------------------------------------------------

## 4. Embedding Videos

Markdown doesn't support direct video embedding,\
but you can use a clickable thumbnail:

[![Video](https://img.youtube.com/vi/dQw4w9WgXcQ/hqdefault.jpg)](https://www.youtube.com/watch?v=dQw4w9WgXcQ)

------------------------------------------------------------------------

## 5. Code Snippets

### Inline Code

Use backticks: `console.log("Hello World")`

### Code Block

\`\`\`javascript 
function greet() { console.log("Hello Markdown!"); }
greet(); 
\`\`\`

-   Banana\
-   Mango

### Nested List

-   Fruits
    -   Apple\
    -   Orange\
-   Vegetables
    -   Carrot\
